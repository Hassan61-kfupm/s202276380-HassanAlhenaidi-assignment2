# My Portfolio

Simple portfolio with About, Projects, and Contact sections.

## How to Run
1. Download files
2. Open index.html with any web browser

## Features
- Smooth scrolling
- Time greeting
- Simple contact form
- Project search
- Dark/Light mode
- Hover animations
